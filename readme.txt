Run from project root.

1) Generate bytecode (baseline + optimized) for CHA, RTA, VTA:
    ./run_all_algorithms.sh Test1

    This creates:
    - sootOutput/Test1/CHA/before/Test.class
    - sootOutput/Test1/CHA/after/Test.class
    - sootOutput/Test1/RTA/before/Test.class
    - sootOutput/Test1/RTA/after/Test.class
    - sootOutput/Test1/VTA/before/Test.class
    - sootOutput/Test1/VTA/after/Test.class

2) Run baseline vs optimized manually (example: CHA):
    java -cp sootOutput/Test1/CHA/before Test
    java -cp sootOutput/Test1/CHA/after Test

3) Compare baseline vs optimized runtime manually:
    /usr/bin/time -f "baseline %e s" java -cp sootOutput/Test1/CHA/before Test >/dev/null
    /usr/bin/time -f "optimized %e s" java -cp sootOutput/Test1/CHA/after Test >/dev/null

4) Repeat for all algorithms:
    for algo in CHA RTA VTA; do
      echo "=== $algo ==="
      /usr/bin/time -f "baseline %e s" java -cp sootOutput/Test1/$algo/before Test >/dev/null
      /usr/bin/time -f "optimized %e s" java -cp sootOutput/Test1/$algo/after Test >/dev/null
    done

5) Inspect generated bytecode (optional):
    javap -classpath sootOutput/Test1/CHA/before -c Test
    javap -classpath sootOutput/Test1/CHA/after -c Test
